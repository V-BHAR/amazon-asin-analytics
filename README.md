# Amazon ASIN Intelligence & Product Analytics Platform

A premium enterprise-level web application for extracting and analyzing Amazon product data from ASINs.

## Features

- **Batch Processing**: Process up to 50,000 ASINs efficiently
- **Real-time Dashboard**: Live progress tracking with analytics
- **Smart Extraction**: Select specific fields to extract for optimal performance
- **Multiple Export Formats**: XLSX, CSV with filtering options
- **Advanced Scraping**: Playwright-based with anti-detection measures
- **Modern UI**: Glassmorphism design with neon effects
- **Docker Support**: Easy deployment with Docker Compose
